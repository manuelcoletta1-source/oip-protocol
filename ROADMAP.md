# OIP Roadmap

## v0.1 (Current Draft)
- Deterministic canonical JSON
- Ed25519 signature
- SHA-256 hashing
- Chain linkage rules
- Reference emitter/verifier
- Conformance suite

---

## v0.2 (Planned)
- Multi-key OID support (key rotation)
- Explicit revocation proof structure
- Transparency log integration model
- Formal RFC 8785 alignment statement
- Strict conformance mode flag

---

## v0.3 (Future)
- Hardware-backed key recommendations
- Attestation extension (optional module)
- Cross-OID interaction events
- Event compression profile
- Streaming verifier mode

---

## Non-Goals

The following are intentionally excluded:
- Policy enforcement rules
- Registry governance
- Network topology
- Blockchain dependency
- Commercial certification models

OIP remains a minimal attribution protocol.
