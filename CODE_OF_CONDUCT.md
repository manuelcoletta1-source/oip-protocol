# Code of Conduct

OIP is a technical specification project.

All contributors are expected to:

- Communicate respectfully and professionally.
- Focus on technical merit.
- Avoid marketing, ideological, or unrelated discussions.
- Provide evidence and reproducible examples when raising issues.

This repository exists for engineering clarity, not opinion debate.

Harassment, personal attacks, or off-topic discussions may result in removal of access.

This project follows a neutral, vendor-agnostic approach.
